## Colors

### Primary

<!--
Start here. Everything else derives from this.
- If you have a brand color, use it.
- If not, pick based on personality:
  Professional → deep blue/slate/indigo
  Friendly → teal/coral/warm purple
  Minimal → black or single muted accent
  Bold → vivid blue/purple/orange
  Technical → gray-blue/green
-->

| Token | Value | Usage |
|-------|-------|-------|
| `color-primary` | `[hex]` | Primary actions, links, focus rings |
| `color-primary-hover` | `[10% darker]` | Hover state for primary |
| `color-primary-light` | `[90% lighter]` | Backgrounds, highlights, selected rows |

### Neutrals

<!--
Tint your grays with your primary hue. Pure gray looks dead.
Warm primary (red/orange/yellow) → warm grays. Cool primary (blue/green/purple) → cool grays.
-->

| Token | Value | Usage |
|-------|-------|-------|
| `color-background` | `[hex]` | Page background |
| `color-surface` | `[hex]` | Card/panel backgrounds |
| `color-border` | `[hex]` | Default borders, dividers |
| `color-text` | `[hex]` | Primary text (≥4.5:1 contrast on background) |
| `color-text-secondary` | `[hex]` | Muted text, labels (≥3:1 on background) |
| `color-text-inverse` | `[hex]` | Text on primary-colored backgrounds |

### Semantic

<!--
Use standard hues (green/amber/red) but match the saturation to your palette.

-->

| `color-warning` | `[hex]` | Warnings, caution |
| `color-error` | `[hex]` | Errors, destructive actions |

## How to Use

### In ASCII Mockups (Feature Specs)
```
┌─ Card (bg: surface, radius: md, shadow: sm) ──────────────┐
│  Title (text: lg, weight: semibold, color: text)           │
│  Description (text: base, color: text-secondary)           │
│  [Button (bg: primary, radius: sm)]                        │
└────────────────────────────────────────────────────────────┘
```

### In Implementation (Tailwind)
```jsx
<div className="bg-surface rounded-md shadow-sm p-4">
  <h2 className="text-lg font-semibold text-gray-900">Title</h2>
  <p className="text-base text-gray-500">Description</p>
  <button className="bg-primary hover:bg-primary-hover text-white rounded-sm px-4 py-2">
    Action
  </button>
</div>
```

### In Implementation (CSS Variables)
```css
:root {
  --color-primary: [hex];
  --spacing-4: [value];
  --radius-md: [value];
}
```
